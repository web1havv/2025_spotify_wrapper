### Manual Setup

**Prerequisites:**
- Node.js 18+
- npm or yarn

**Installation:**

1. Install backend dependencies
```bash
cd backend
npm install
```

2. Install frontend dependencies
```bash
cd ../frontend
npm install
```

3. Start the backend server
```bash
cd backend
npm run dev
```

4. Start the frontend (in a new terminal)
```bash
cd frontend
npm run dev
```
